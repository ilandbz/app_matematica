Ejemplos de Propiedades de Potenciación

Propiedad de Multiplicación de Potencias con la misma base:

Para calcular a/b * c/d, donde "a", "b", "c" y "d" son números enteros, simplemente multiplicamos las bases y los exponentes:

a/b * c/d = a*c/b*d

Propiedad de División de Potencias con la misma base:

Para calcular a/b / c/d, donde "a", "b", "c" y "d" son números enteros, simplemente dividimos las bases y restamos los exponentes:

a/b / c/d = a*c/b*d

Propiedad de Potencia de una Potencia:

Para calcular (a/b)c, donde "a", "b" y "c" son números enteros, elevamos la base a la potencia "c":

(a/b)c = ac/bc

Propiedad de Potencia de un Producto:

Para calcular (a/b * c/d)e, donde "a", "b", "c", "d" y "e" son números enteros, elevamos cada factor a la potencia "e":

(a/b * c/d)e = (a/b)e * (c/d)e

Ejemplos de Propiedades de Radicación

Propiedad de Multiplicación de Raíces con la misma base:

Para calcular √a * √b, simplemente multiplicamos las raíces:

√a * √b = √(a * b)

Propiedad de División de Raíces con la misma base:

Para calcular √a / √b, simplemente dividimos las raíces:

√a / √b = √(a / b)

Propiedad de Potencia de una Raíz:

Para calcular (√a)^n, elevamos la raíz a la potencia "n":

(√a)^n = √(a^n)

Propiedad de Raíz de un Producto:

Para calcular √(a * b), simplemente tomamos la raíz de cada factor y luego multiplicamos:

√(a * b) = √a * √b

Ejemplos de Ecuaciones Exponenciales

Ejemplo 1:

Resolver la ecuación exponencial: 2^x = 16

Solución:

x = log2(16)

x = 4

Ejemplo 2:

Resolver la ecuación exponencial: 3^(x-1) = 9

Solución:

x - 1 = log3(9)

x - 1 = 2

x = 3

Ejemplo 3:

Resolver la ecuación exponencial: 5^(2x+1) = 125

Solución:

2x + 1 = log5(125)

2x + 1 = 3

2x = 2

x = 1
